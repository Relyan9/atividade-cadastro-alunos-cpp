Projeto acadêmico desenvolvido em C++ para cadastrar alunos, calcular médias e exibir os resultados em formato de tabela.

- Funcionalidades
Cadastro de 5 alunos;
Leitura do nome e de 3 notas de cada aluno;
Cálculo da média por meio de uma função;
Verificação da situação do aluno (Aprovado/Reprovado);
Identificação do aluno com a maior média;
Exibição dos resultados em formato de tabela.

- Tecnologias
C++
iostream
iomanip

Este projeto foi desenvolvido como atividade acadêmica para praticar conceitos fundamentais da linguagem C++, como:

- Funções;
Vetores e matrizes;
Estruturas de repetição;
Estruturas condicionais;
Entrada e saída de dados.
